# 📅 Leave Tracker — ระบบบันทึกวันลาส่วนตัว

พัฒนาโดย Sarawut.C | สำหรับใช้ส่วนตัวเท่านั้น

## ไฟล์ในโปรเจกต์

| ไฟล์ | หน้าที่ |
|------|---------|
| `Code.gs` | Backend ทั้งหมด (บันทึก/แก้ไข/ลบ/โบนัส) |
| `Sidebar.html` | UI หลัก (Sidebar 4 แท็บ) |
| `SettingsDialog.html` | หน้าตั้งค่าสิทธิ์วันลา |

## วิธีติดตั้งใช้งาน

1. เปิด Google Sheets
2. ไปที่ **Extensions → Apps Script**
3. สร้างไฟล์ตามชื่อข้างบน แล้ววางโค้ดจาก GitHub
4. **Save** → กลับ Sheets แล้ว **Refresh**
5. คลิกเมนู 📅 บันทึกวันลา → 🔧 สร้างหัวตารางใหม่

## ฟีเจอร์

- ✅ บันทึกวันลา (เต็มวัน / ครึ่งวัน / รายชั่วโมง)
- ✅ สรุปวันลาคงเหลือ
- ✅ ประวัติการลา 200 รายการ (แก้ไข/ลบได้)
- ✅ คำนวณโบนัสประจำปี
- ✅ ตั้งค่าสิทธิ์วันลาใน Sidebar
